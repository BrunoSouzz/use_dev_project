# 📱 UseDev

**UseDev** é um aplicativo mobile desenvolvido como projeto acadêmico, com o objetivo de oferecer uma plataforma de e-commerce voltada especialmente para programadores.

A proposta do app é conectar desenvolvedores a produtos úteis no dia a dia, como cursos, ferramentas, acessórios e recursos tecnológicos.

---

## 🚀 Tecnologias Utilizadas

* **Dart**
* **Flutter**
* **Android Studio**

---

## 📦 Funcionalidades

* 🛍️ Catálogo de produtos voltados para programadores
* 🔍 Sistema de busca de produtos
* 🛒 Carrinho de compras
* 👤 Sistema básico de usuário
* 💳 Simulação de checkout

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com fins acadêmicos, visando:

* Praticar o desenvolvimento mobile com Flutter
* Aplicar conceitos de UI/UX
* Simular um sistema real de e-commerce
* Trabalhar integração de funcionalidades comuns em apps comerciais
